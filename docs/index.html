MM-CompMatrix2

MillMage CNC Compatibility Checker — Documentation

Overview

A single-page web app that helps CNC users check whether their machine is compatible with MillMage software. Two views:

Public View

Read-only search and browsable table. Users search by machine brand/model and see a compatibility detail card.

Admin View

Password-protected. Add, edit, delete machines. CSV and JSON import/export. Changes persist in browser localStorage.

Architecture

Tech Stack

  • Vite 6 + React 18 — build and UI framework
  • Static JSON — default machine data (public/data/machines.json)
  • localStorage — persists admin edits client-side
  • Inline styles — no CSS framework, ported from Claude WebUI artifacts

Deployment

Vercel (Production)

  • Push to GitHub → auto-deploy
  • Set VITE_ADMIN_PASSWORD in Vercel Environment Variables
  • Password baked into JS at build time

Local Development (no Node.js needed)

./docker-dev.sh                           # http://localhost:5173
ADMIN_PASSWORD=secret ./docker-dev.sh     # custom password

Runs Vite dev server inside a Docker container with hot-reload. Alternatively, if you have Node.js installed:

npm install && npm run dev

Docker

No host Node.js required. All build and dev workflows run inside Docker containers.

Build (manual)

docker build --platform linux/amd64 \
  -t mm-compmatrix:0.1.0 \
  -t mm-compmatrix:latest .

docker save -o mm-compmatrix.tar \
  mm-compmatrix:0.1.0 mm-compmatrix:latest

Run

docker run -d -p 8080:80 \
  -e ADMIN_PASSWORD=mysecret \
  mm-compmatrix:latest

Access at http://localhost:8080

Password Injection

The docker-entrypoint.sh generates /usr/share/nginx/html/env.js at container start, injecting the ADMIN_PASSWORD env var as window.__ENV__.ADMIN_PASSWORD.
